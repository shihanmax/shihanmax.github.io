# 目录树优化说明

## 概述

根据 `docs/development.md` 中的开发需求，对博客的目录树功能进行了全面优化，实现了以下特性：

## 主要改进

### 1. 独立悬浮面板设计

- **位置**: 目录面板固定在页面左侧，与 `post-container` 顶部对齐
- **独立性**: 目录面板完全独立于文章内容区域，不影响文章宽度
- **悬浮效果**: 使用 `position: fixed` 实现真正的悬浮效果
- **阴影效果**: 添加了优雅的阴影效果，提升视觉层次

### 2. 展开/收起功能

- **切换按钮**: 右上角的切换按钮支持展开/收起操作
- **状态保持**: 支持折叠状态，收起时显示为窄条
- **动画效果**: 平滑的展开/收起动画过渡
- **图标变化**: 展开时显示 "×"，收起时显示 "≡"

### 3. 层级目录显示

- **多级支持**: 支持 H1-H6 六级标题的层级显示
- **缩进设计**: 不同级别使用不同的左缩进，清晰展示层级关系
- **字体权重**: 不同级别使用不同的字体权重，增强视觉层次
- **字体大小**: 深层级别使用较小的字体，保持整体协调

### 4. 滚动同步功能

- **自动高亮**: 根据当前阅读位置自动高亮对应的目录项
- **平滑滚动**: 点击目录项时平滑滚动到对应章节
- **位置跟踪**: 实时跟踪滚动位置，准确显示当前阅读章节
- **自动滚动**: 当活动项超出视窗时，自动滚动目录以显示活动项

### 5. 基于Markdown原文解析

- **原文解析**: 直接从Markdown原文解析目录，而非HTML
- **锚点生成**: 自动生成URL友好的锚点ID
- **层级构建**: 智能构建层级结构，支持嵌套目录
- **性能优化**: 解析过程高效，支持大型文档

## 技术实现

### 后端实现

1. **TOCParser类** (`utils/toc_parser.py`)
   - 基于正则表达式解析Markdown标题
   - 生成层级目录结构
   - 支持HTML和JSON输出格式

2. **MarkdownParser扩展** (`utils/markdown_parser.py`)
   - 集成目录解析功能
   - 提供便捷的API接口
   - 支持目录摘要信息

3. **API端点** (`app.py`)
   - `/api/posts/<year>/<month>/<slug>/toc` - 获取文章目录
   - 支持公开访问，无需认证

### 前端实现

1. **TOCManager类** (`templates/post.html`)
   - 面向对象的目录管理
   - 事件绑定和状态管理
   - 滚动监听和位置更新

2. **CSS样式** (`static/css/main.css`)
   - 独立的悬浮面板样式
   - 响应式设计支持
   - 深色主题适配

3. **响应式设计**
   - 移动端自动隐藏目录面板
   - 桌面端保持完整功能
   - 平滑的布局过渡

1. **节流处理**: 滚动事件使用节流，避免过度触发
2. **缓存机制**: 目录解析结果可缓存，提升性能
3. **懒加载**: 只在需要时生成目录
4. **内存管理**: 合理的事件监听器管理

## 使用说明

### 自动生成

目录树会在页面加载时自动生成，无需手动配置。

### 手动刷新

如果需要手动刷新目录，可以调用：

```javascript
if (window.tocManager) {
    window.tocManager.generateTOC();
}
```

### API调用

获取文章目录数据：

```javascript
fetch('/api/posts/2024/1/article-slug/toc')
    .then(response => response.json())
    .then(data => {
        console.log('目录数据:', data.data.toc);
    });
```

## 配置选项

### 目录面板宽度

在CSS中修改 `.toc-panel` 的 `width` 属性：

```css
.toc-panel {
    width: 320px; /* 调整宽度 */
}
```

### 层级缩进

修改各级别的 `padding-left` 值：

```css
.toc-item.level-2 > .toc-link {
    padding-left: 2rem; /* 调整缩进 */
}
```

## 故障排除

### 目录不显示

1. 检查文章是否包含标题（H1-H6）
2. 确认JavaScript没有错误
3. 检查CSS样式是否正确加载

### 滚动同步问题

1. 确认文章内容区域有正确的ID
2. 检查标题元素是否有正确的ID
3. 验证滚动监听器是否正常工作

### 样式问题

1. 检查CSS文件是否正确加载
2. 确认没有样式冲突
3. 验证响应式断点设置
