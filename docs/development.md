## 1. 大纲（目录树）支持

实现大纲显示功能，要求：

1. 大纲的存在与否 不影响原有页面的位置，布局；
2. 大纲可以支持展开，收起，收起时是一个bar，点击展开后，向下展开为大纲
3. 大纲支持层级目录显示，且需要随着文章位置滚动到对应章节；
4. 点击大纲上的章节名，可以跳转到对应位置；
5. 大纲不仅要支持阅读页，也需要保留对编辑页的右侧预览页的兼容性


我重申一下需求：

1. 目录树的顶部，应和postcontainer顶部对齐，但是不要放在post-container中；
2. 目录树应该是一个独立的，悬浮的面板，贴在页面最左侧，并且底部设置在屏幕底部，且这个container固定，只有内部内容随着文章动；
3. 目录树看是否可以通过md原文而不是html来解析构建？
4. 应该使得文章的post-container保持原来的宽度，并且其内部文字不应该受到侵占；
